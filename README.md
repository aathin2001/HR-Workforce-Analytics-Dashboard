# HR Analytics Dashboard
A Tableau dashboard built to analyze employee workforce data, demographics, and compensation patterns.

## 📊 Overview

### Summary
- Active employees, hired vs terminated counts, and year-over-year hiring trends
- HQ vs branch employee distribution
- Department and job title breakdown

### Demographics
- Gender ratio, age group, and education level distribution
- Correlation between education level and performance rating

### Income
- Salary comparison across education levels and genders
- Age-salary correlation analysis by department

### Employee Records
- Fully filterable record-level view — search by ID, department, location, salary range, status, and experience

## 🛠️ Tools Used
- **Tableau** — dashboard development
- **MySQL** — source data
